# LibSpace Status

This repository contains the status page for [LibSpace](https://libspace.io), powered by [Upptime](https://upptime.js.org).

## Status

Live status page: **https://status.libspace.io**

## Monitored Services

| Service     | URL                                    |
| ----------- | -------------------------------------- |
| LibSpace    | https://libspace.io                    |
| API         | https://api.libspace.io/health         |
| Staging     | https://staging.libspace.io            |
| Staging API | https://api.staging.libspace.io/health |

## How It Works

- GitHub Actions checks service availability every 5 minutes
- Status history and response times are stored in this repository
- The status page is automatically deployed to GitHub Pages
- Incidents are tracked via GitHub Issues
